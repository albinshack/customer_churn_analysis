#  Customer Churn Analysis

##  Project Overview

This project focuses on analyzing customer churn behavior using exploratory data analysis (EDA) and visualization techniques. The goal is to identify key factors influencing churn and provide actionable business recommendations to improve customer retention.

---

##  Dataset Description

The dataset contains customer-related information with the following features:

* **CustomerID** – Unique identifier for each customer
* **Age** – Customer's age
* **Gender** – Male/Female
* **Tenure** – Duration of customer relationship
* **MonthlyCharges** – Monthly subscription cost
* **ContractType** – Type of contract (Monthly, Yearly, etc.)
* **InternetService** – Type of internet service
* **TotalCharges** – Total amount charged
* **TechSupport** – Availability of tech support
* **Churn** – Whether the customer churned (1 = Yes, 0 = No)

---

##  Exploratory Data Analysis (EDA)

Key steps performed:

* Handling missing values (InternetService column)
* Data cleaning and preprocessing
* Univariate and bivariate analysis
* Visualization using bar charts, histograms, and box plots

---

##  Key Insights

* **High Churn Rate**: Approximately **88.3%** of customers have churned
* **Contract Type Impact**: Month-to-month contracts show the highest churn (nearly 100%)
* **Pricing Sensitivity**: Customers with higher monthly charges are more likely to churn
* **Customer Tenure**: New customers are more likely to leave

---

##  Recommendations

* Offer discounts and incentives for long-term contracts
* Improve onboarding experience for new customers
* Provide loyalty rewards for high-value customers
* Introduce retention strategies for month-to-month users
* Reevaluate pricing to align with customer expectations
