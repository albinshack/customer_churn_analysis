Customer Churn Analysis

Objective
Analyze customer behavior to identify churn patterns and provide actionable business recommendations.

Tools Used
Python (pandas, matplotlib, sklearn)

Key Insights
• Month-to-month contracts have highest churn rate
• High monthly charges increase churn risk
• New customers are more likely to leave

Business Recommendations
• Promote long-term contracts
• Improve onboarding experience
• Offer retention incentives

Outcome
This project demonstrates the ability to analyze customer data and generate actionab
